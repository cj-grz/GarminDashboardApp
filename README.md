A custom dashboard-style watch face for Garmin devices built using the Connect IQ SDK and Monkey C.
Features

Circular dashboard layout with 8 independent metric gauges

Real-time metric retrieval using Garmin Connect IQ APIs

Color-coded neon visual system for quick differentiation

Centered digital clock with date display

Modular architecture separating:

UI rendering

metric retrieval

drawing logic

theme configuration

Designed specifically for high-resolution round displays such as the Fenix 7X Pro
